# Standalone Project Status

`structured-latent-text-refinement/` is the official standalone project.

Repository:

```text
https://github.com/saslifat-gif/structured-latent-text-refinement
```

If Git needs to be initialized manually from a shell, run:

```bash
cd structured-latent-text-refinement
git init
git branch -M main
git remote add origin git@github.com:saslifat-gif/structured-latent-text-refinement.git
git add .
git commit -m "Create standalone structured latent text refinement project"
git push -u origin main
```

The parent `ml-foundations` workspace is now archival context. Current project
work should happen inside this directory.
